# Detect cycle in an directed graph

## Problem Description
Given a Directed Graph with V vertices (Numbered from 0 to V-1) and E edges, check whether it contains any cycle or not.

**Example 1:**
Input:  V = 4, adj = {{1}, {2}, {3}, {3}} 
Output: 1

**Example 2:**
Input: V = 3, adj = {{1}, {2}, {}}
Output: 0
